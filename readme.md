# Git Deploy 

Simple bash script to deploy from git


## Make Script Executable 


```
chmod u+x git_deploy.sh
```

## Usage

```

./git_deploy.sh
Git status will run, will ask if the git status looks good.
If yes, type yes. 
```
The following will run. 
```

-git add. 
-git commit -m "deploy"
-git push
```

If not, the script will exit.
